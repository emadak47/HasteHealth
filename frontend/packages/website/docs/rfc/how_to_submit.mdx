# How to Submit an RFC
1. When submitting an RFC, please follow the format outlined in the [RFC Format](/docs/rfc/format) document.
2. Open a PR with a new document in the [RFC Proposals directory](https://github.com/HasteHealth/HasteHealth/tree/main/frontend/packages/website/docs/rfc/proposals).
3. Once the PR is submitted, request reviews from code owners.
